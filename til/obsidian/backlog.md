- [x] [PKM](til/obsidian/pkm.md) - 개인 지식 관리(Personal Knowledge Management)의 기본 개념과 목적

## 핵심 개념
- [x] [Vault](til/obsidian/vault.md) - Obsidian의 기본 저장 단위. 로컬 파일시스템 기반 폴더 구조
- [x] [Wikilink와 Backlink](til/obsidian/wikilink-backlink.md) - `[[이중 대괄호]]`로 노트 간 양방향 연결을 만드는 핵심 메커니즘
- [x] [Properties](til/obsidian/properties.md) - YAML 기반 노트 메타데이터 시스템. 타입 지정, 검색/필터에 활용
- [x] [Graph View](til/obsidian/graph-view.md) - 노트 간 연결을 시각적 그래프로 보여주는 기능
- [x] [Core Plugins](til/obsidian/core-plugins.md) - 내장 플러그인 (Command Palette, Daily Notes, Templates, Bookmarks 등)
- [x] [검색과 태그](til/obsidian/search-and-tags.md) - 노트 검색, `#태그` 시스템, 필터링 방법
- [x] [Callout과 Embed](til/obsidian/callout-embed.md) - `> [!callout]` 블록과 `![[노트]]` 임베딩 문법
- [x] [Map of Content](til/obsidian/map-of-content.md) - 노트들을 체계적으로 조직하는 허브 노트(MOC) 패턴
- [x] [Zettelkasten](til/obsidian/zettelkasten.md) - 원자적 노트와 연결 기반 지식 관리 방법론

## 심화
- [x] [Community Plugins](til/obsidian/community-plugins.md) - 커뮤니티 플러그인 생태계 (Dataview, Templater, Calendar 등)
- [x] [Dataview](til/obsidian/dataview.md) - SQL 유사 쿼리로 노트를 데이터처럼 조회하는 플러그인
- [x] [Templater](til/obsidian/templater.md) - 동적 변수와 JavaScript를 지원하는 고급 템플릿 플러그인
- [x] [Obsidian Git](til/obsidian/obsidian-git.md) - Obsidian 내에서 Git 자동 커밋/동기화. GitHub 연동과 버전 관리
- [x] [Canvas](til/obsidian/canvas.md) - 무한 화이트보드. 노트, 이미지, 카드를 자유 배치하는 시각적 도구
- [x] [Bases](til/obsidian/bases.md) - 노트를 데이터베이스 뷰로 편집/정렬/필터링하는 기능 (2025년 출시)
- [x] [Obsidian Sync](til/obsidian/obsidian-sync.md) - E2E 암호화 기반 디바이스 간 vault 동기화 서비스
- [x] [Obsidian Publish](til/obsidian/obsidian-publish.md) - 선택한 노트를 웹사이트로 공개하는 퍼블리싱 서비스
- [x] [Plugin 개발](til/obsidian/plugin-development.md) - TypeScript API, Plugin 클래스 라이프사이클, esbuild 빌드 시스템
- [ ] [CSS Snippets과 테마](til/obsidian/css-snippets-themes.md) - CSS를 통한 외관 커스터마이징과 커뮤니티 테마 적용
- [ ] [Hotkeys 커스터마이징](til/obsidian/hotkeys.md) - 단축키 설정과 hotkeys.json 직접 편집으로 워크플로우 최적화
- [x] [Obsidian CLI](til/obsidian/obsidian-cli.md) - 터미널에서 Obsidian vault를 조작하는 공식 CLI 도구
